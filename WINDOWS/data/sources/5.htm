Para terminar


Ojalá que haya disfrutado con este pequeño viaje por el cosmos. Si es así por favor visita la web de Miguel Bautista, el creador de este proyecto (www.m40s.info).

Este proyecto es de Software Libre, por lo cual no obtiene ningún apoyo económico por su distribución. Si quiere apoyar el proyecto para que se continue actualizando y mejorando puede hacer dos cosas:

  • Difundirlo, ya sea por redes sociales, recomendandoselo a la gente que conoce, et cetera.
  • También puede realizar una donación mediante PayPal, BitCoin o DogeCoin. En el archivo README puede encontrar más información sobre las donaciones. Cualquier donación, por pequeña que sea es de gran ayuda.

Puede encontrar el código de este programa, página de descarga y versiones antiguas en www.m40s.info.

¡Gracias por utilizar este programa!
